# 05 - Counters & Registers

计数器与寄存器的 Verilog 实现。

## 待实现

| 模块 | 描述 | 状态 |
|------|------|------|
| 4-bit binary counter | 带使能的 4 位二进制计数器 | ⬜ |
| BCD counter | 0-9 BCD 计数器 | ⬜ |
| up/down counter | 可逆计数器 | ⬜ |
| shift register (SISO) | 串入串出移位寄存器 | ⬜ |
| shift register (PIPO) | 并入并出移位寄存器 | ⬜ |
| universal shift register | 通用移位寄存器 | ⬜ |
| LFSR | 线性反馈移位寄存器 | ⬜ |

## 知识点

- 同步 vs 异步计数器
- 时序逻辑中的复位策略
- 移位寄存器的串/并转换
- LFSR 用于伪随机数生成和 CRC
